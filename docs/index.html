FAQ


Для каких роле подходит?

Backend, Fullstack, Frontend, DevOps, Data Engineer, ML Engineer и любые другие IT-позиции. Бот адаптирует вопросы под то что ты напишешь.

Насколько вопросы реалистичны?

Вопросы генерирует Claude — одна из лучших моделей. Они соответствуют реальным вопросам на собеседованиях в IT-компаниях: поведенческие по STAR, технические по твоему стеку и грейду.

Что такое Telegram Stars?

Внутренняя валюта Telegram. Покупается прямо в приложении через App Store или Google Play. Никаких карт и KYC не нужно.

Мои данные в безопасности?

Тексты ответов отправляются в Anthropic API для обработки. Никаким другим третьим лицам данные не передаются. Удалить всё можно командой /deletedata в боте.

Можно ли готовиться к конкретной вакансии?

Да — при настройке сессии вставь текст вакансии и бот адаптирует вопросы под требования конкретной позиции.
